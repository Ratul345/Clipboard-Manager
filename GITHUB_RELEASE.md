# 🎉 Clipboard Manager v1.0.0 - Initial Release

A lightweight, cross-platform clipboard manager that saves your copy history and lets you reuse it anytime.

## ✨ Features

- 📋 **Automatic Clipboard Monitoring** - Captures everything you copy (text, links, images)
- 🔍 **Fast Search** - Quickly find items with real-time search filtering
- 🖼️ **Image Support** - Stores and displays copied images
- 🔗 **Smart Link Detection** - Automatically identifies and highlights URLs
- 🎨 **Clean UI** - Modern, intuitive interface with item cards
- 🔔 **System Tray Integration** - Runs quietly in the background
- ⌨️ **Keyboard Navigation** - Arrow keys, Enter, Delete, and Escape shortcuts
- 🗑️ **Easy Management** - Delete individual items or clear all history
- 💾 **Persistent Storage** - SQLite database keeps your history across sessions
- 🖥️ **Cross-Platform** - Works on Windows and Linux (X11/Wayland)

## 📥 Download

### Windows (Portable)
Download `ClipboardManager-v1.0.0-Portable.exe` below - no installation required!

**How to use:**
1. Download the .exe file
2. Double-click to run
3. Look for the clipboard icon in your system tray
4. Press **Ctrl+Shift+V** to open the clipboard manager

**Note:** Windows may show a SmartScreen warning because the app is not code-signed. Click "More info" → "Run anyway" to proceed.

## 🚀 Quick Start

1. **Copy something** - Copy any text, link, or image (Ctrl+C)
2. **Open the manager** - Press **Ctrl+Shift+V** or click the system tray icon
3. **Find your item** - Use the search bar or scroll through history
4. **Reuse it** - Click an item or press Enter to copy it back to clipboard
5. **Paste** - Use Ctrl+V to paste anywhere

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| **Ctrl+Shift+V** | Open/close Clipboard Manager |
| **↑ / ↓** | Navigate through items |
| **Enter** | Copy selected item and close |
| **Delete** | Remove selected item |
| **Escape** | Close window |
| **Ctrl+F** | Focus search bar |

## 📋 System Requirements

- **Windows:** Windows 10 or later
- **Linux:** Ubuntu 20.04+, Fedora 35+, or similar (X11/Wayland)
- **RAM:** 100 MB minimum
- **Disk Space:** 50 MB for application + storage for clipboard history

## 🐛 Known Issues

- On first run, Windows Defender may scan the file (this is normal)
- Global hotkey may not work if another app uses Ctrl+Shift+V
